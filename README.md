## Elementos de juego
Los tres jugadores deben trabajar en equipo para completar cada uno de los tres niveles. Cada nivel se gana llegando a la moneda
Multijugador: para exactamente 3 jugadores
Tipo de juego: puzzle
Vista: isométrica
